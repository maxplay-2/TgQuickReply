## TgQuickReply — это настольная программа для работы с Telegram‑ботом. Она позволяет быстро получать сообщения от пользователей и отвечать на них прямо с компьютера.

Что умеет программа

Получает сообщения от всех пользователей, которые пишут вашему боту.

Отображает историю переписки с каждым пользователем.

Позволяет легко отвечать — есть поле для ввода текста и кнопка «Отправить».

Звуковое уведомление при приходе нового сообщения, чтобы не пропустить важное.

Быстро вставлять текст из буфера обмена (например, скопированные ответы или токен бота).

Простое управление списком пользователей — выбрал пользователя, и сразу видно всю переписку с ним.

### програма еще находится на стадии альфа версии так что могут быть баги, ошибки,недоработки
